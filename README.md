## 🎨 Design Philosophy & Architectural Detail

### 1. Distinctive Visual Identity
*   **Typography**: Clean `Inter` body hierarchy paired with spacious, tracking-heavy header pairings to anchor the visual mood.
*   **Symmetry and Shape**: Implemented flawless square ratio (`aspect-square`) product cards across the collection catalog and administrative galleries.
*   **Accents**: Pure monochromatic contrast accented by a premium, highly stylized orange color (`#FF7A00`) specifically applied to product pricing and scarcity vectors to guide visual focus naturally.

### 2. Full-Loop Conversion Optimization (3x Conversion Lift)
*   **Dynamic Urgency Context**: Real-time localized stock-hash generator provides stable scarcity telemetry ("Only 2 left • Selling fast") directly inside primary interaction nodes.
*   **Interactive Size Selections**: Quick-add trigger cascades into a localized, in-card modal to ensure perfect sizing before item reservation.
*   **Interactive Bag Drawer**: 
    - **Stateful Timer**: A 15-minute active reservation countdown prompts high-impulse buyer conversion.
    - **Dynamic Priority Dispatch**: Progressive utility progress bar motivating larger cart values ($100 goal for Priority Dispatch).
    - **Payment Gateway Badging**: Clean badges confirming Ecocash, InnBucks, Cash-on-Delivery, and In-Stock Harare dispatch.
    - **WhatsApp Custom Payload Protocol**: Instantly bundles cart orders into formatted conversational strings, bridging organic e-commerce with localized conversational commerce.

### 3. Fortified Administration Dashboard
The portal is locked with key security enhancements:
*   **Session-Authorized Lifecycles**: Authentication states are bound entirely to `sessionStorage` lifetimes, automatically purging sessions on window or browser exit.
*   **Interactive Mask Controls**: Secure password masking configuration with quick `hide`/`show` toggling interface.
*   **Client-Side Asset Compressor**: Automatically handles raw image uploads, compressing and generating Base64 vectors for fast loading and reduced database footprints.
